# 2.1-.-Machine-Learning-Supervised-Learning-
Multiple Linear Regression: Startup Profit Prediction

ML project predicting startup profits using R&D, Administration, Marketing spends, and location. Handles categorical data with one-hot encoding. Full pipeline from preprocessing to deployment with 98% accuracy.

Files:

1).Multiple_Linear_Regression(Salary_Prediction).ipynb - Training & Analysis

2).Deployment_Multiple_Linear_Regression.ipynb - Prediction Script

3).50_Startups.csv - Dataset

4).Finalized_model_MLR_profit_prediction.sav - Trained Model


Tech: Python, Pandas, Scikit-learn, One-Hot Encoding

Quick Start: Run deployment notebook and input spending values to predict profit.
